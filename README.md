# iceui-weapp

由于小冰天气自定义组件越写越多, 索性就把它们抽离出来了 (～￣▽￣)～

## 启动本地开发

```bash
npm run dev
```

可以通过传递 `--open` 指令自动打开小程序开发者工具
```bash
npm run dev:open
```

## 构建打包

```bash
npm run build
```
