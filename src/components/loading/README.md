# cell 组件

## 使用

```html
<ice-loading type="normal"></ice-loading>

<ice-loading type="shutter"></ice-loading>

<ice-loading type="shutter-2"></ice-loading>

<ice-loading type="shutter-3"></ice-loading>

<ice-loading type="shutter-4"></ice-loading>

<ice-loading type="pulse-1"></ice-loading>

<ice-loading type="pulse-2"></ice-loading>

<ice-loading type="pulse-3"></ice-loading>
```

### 设置大小

```html
<ice-loading type="normal" size="40"></ice-loading>
```

### 添加文字
```html
<ice-loading type="normal" text="加载中"></ice-loading>

<!-- 设置文字垂直分布 -->
<ice-loading type="normal" text="加载中" vertical></ice-loading>
```
