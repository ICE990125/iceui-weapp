# button 组件

## 使用

### outlined

```html
<ice-button outlined></ice-button>
```

### flat
```html
<ice-button flat></ice-button>
```

### 圆角
```html
<ice-button rounded></ice-button>
```

### 图标
```html
<ice-button icon="ri ri-bar-chart-fill"></ice-button>
```

### 图标加文字
```html
<ice-button icon="ri ri-bar-chart-fill">排行榜</ice-button>
```
